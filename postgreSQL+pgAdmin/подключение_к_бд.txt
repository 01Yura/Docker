Данный docker-compose файл запускает БД postgreSQL и pgAdmin для подключения к ней.
Чтобы подключится к БД через pgAdmin, в браузере нужно войти в сам pgAdmin введя в адресной строке браузера:
localhost:8082
Затем ввести
Email: noemail@noemail.com
Password: toor

Чтобы подключится к самой postgresql, нужно нажать "добавить новый сервер"
Имя: любое
Имя/адрес сервера: postgresql (это имя контейнера, так как docker-compose использует имя контейнера как DNS имя)
Имя пользователя: root
Пароль: toor

Чтобы добавить таблицы (.sql) через командную строку:
- подключится к контейнеру
    docker exec -it ff32cf4c733e bash
- запустить psql с аргументами
    psql -U root -d test_db -f path/to/file.sql #нужно чтобы база данных сущуствовала, если ее нет, то надо создать через графический интерфейс или через 
    docker exec -it ff32cf4c733e psql -U root -d postgres -c "CREATE DATABASE test_db" (после -d надо указать существующую БД)

В PostgreSQL должна существовать как минимум одна база данных, к которой можно подключиться для выполнения команд создания новых баз данных. Обычно это база данных по умолчанию postgres, которая создается при установке PostgreSQL.
